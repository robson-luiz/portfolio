## 🌟 About

Professional portfolio developed to showcase my projects and skills as a Freelance Web Developer, focused on building custom websites, web systems, and landing pages for businesses.

## 🚀 Features

- **Responsive Design** - Adapts to different screen sizes
- **Light/Dark Mode** - Smooth theme switching
- **Multi-language** - Support for Portuguese and English
- **Services Offered**:
  - Professional Websites - Fast, responsive websites optimized for SEO
  - Web Systems - Custom systems to automate and organize businesses
  - Landing Pages - Conversion-focused pages
  - Maintenance & Support - Ongoing updates and improvements
- **Featured Projects**:
  - FM System - Personal financial management system with Laravel 12, Tailwind CSS and MySQL
  - Personal Blog - Blog in PHP with admin panel, developed in 2017 and refactored in 2025
  - Celke Website - New version of the Celke website and platform, built with Laravel 12 and Tailwind CSS
  - SimplePage - Website with administrative system in PHP and Bootstrap, developed in 2017 and refactored in 2025

## 🛠️ Technologies Used

- **Frontend**:
  - HTML5

  - Font Awesome (icons)

- **Tools**:
